### Step 2. Replace `operation_reserch` -> `operation_research` in all HTML files

Scope: all `*.html` files in the project.

Pattern: exact substring `operation_reserch` -> `operation_research`.

This is safe because:
- the string `operation_reserch` only appears in URLs and file references;
- `systems_research.html` does not match this pattern;
- PDF assets like `operation-research_akoff.pdf` use a hyphen and do not match;
- no content text uses the phrase `operation_reserch`.

This covers all of the following in one pass:
- canonical URLs
- hreflang links
- og:url meta tags
- JSON-LD fields (@id, url, mainEntityOfPage)
- navigation links
- breadcrumb links
- cross-article text-links
- EN<->RU language switcher links

### Step 3. Add 301 redirects to `.htaccess`

Add `RewriteEngine On` and 18 `RewriteRule` entries (9 RU + 9 EN) for 1:1 permanent redirects.

Format:
```
RewriteRule ^operation_reserch\.html$ /operation_research.html [R=301,L]
RewriteRule ^en/operation_reserch\.html$ /en/operation_research.html [R=301,L]
```
