Definiciones de sistema

Existen varias decenas de definiciones de este concepto. La definición del concepto de sistema ha cambiado no solo en su forma, sino también en su contenido.

Sistema:

  • complejo de componentes en interacción (L. von Bertalanffy).
  • conjunto de elementos que se encuentran en determinadas relaciones entre sí y con el entorno (L. von Bertalanffy).
  • un todo compuesto de muchas partes. Ensamblaje de características (C. Cherry).
  • conjunto de elementos interrelacionados, diferenciado del entorno e interactuante con él como un todo (F. I. Peregúdov, F. P. Tarásenko).
  • disposición, conjunto o colección de cosas vinculadas o relacionadas entre sí de tal modo que juntas forman una cierta unidad, una integridad; disposición de componentes físicos vinculados o relacionados entre sí de tal modo que forman o actúan como una unidad íntegra (Distefano).
  • combinación de elementos en interacción, organizados para la consecución de uno o varios objetivos fijados (ISO/IEC 15288).
  • conjunto finito de elementos funcionales y de relaciones entre ellos, delimitado del entorno en correspondencia con un determinado objetivo dentro de un determinado intervalo temporal (V. N. Sagatovski).
  • reflejo en la conciencia del sujeto (investigador, observador) de las propiedades de los objetos y sus relaciones en la resolución del problema de investigación y conocimiento (Yu. I. Chernyak).
  • el sistema S en el objeto A respecto a la propiedad (cualidad) integrativa es el conjunto de aquellos elementos que se encuentran en tales relaciones que generan dicha propiedad integrativa (E. B. Agoshkova, B. V. Ajlibininski).
  • conjunto de elementos integrados e interactuantes de forma regular o interdependientes, creado para la consecución de determinados objetivos, donde las relaciones entre los elementos están definidas y son estables, y la productividad o funcionalidad global del sistema es superior a la simple suma de sus elementos (PMBOK).
  • dispositivo que acepta una o más entradas y genera una o más salidas (Drenik).
  • dispositivo, proceso o esquema que se comporta según una determinada prescripción; la función del sistema consiste en operar en el tiempo con información y/o energía y/o materia para producir información y/o energía y/o materia (D. Ellis, F. Ludwig).
  • abstracción matemática que sirve de modelo de un fenómeno dinámico (G. Freeman).
  • conjunto integrado de elementos en interacción, destinado al cumplimiento cooperativo de una función previamente definida (R. Gibson).
  • conjunto de objetos junto con las relaciones entre los objetos y entre sus atributos (A. Hall, R. Fagen).
  • colección de entidades o cosas, animadas o inanimadas, que percibe ciertas entradas y actúa conforme a ellas para producir determinadas salidas, persiguiendo el objetivo de maximizar ciertas funciones de las entradas y las salidas (R. Kershner).
  • región delimitada en el espacio y en el tiempo, en la que las partes-componentes están unidas por relaciones funcionales (J. Miller).
  • desde el punto de vista matemático, una cierta parte del mundo que en cualquier momento dado puede describirse asignando valores concretos a un cierto conjunto de variables; no es simplemente un conjunto de unidades (partículas, individuos), cada una de las cuales se rige por leyes de causalidad que actúan sobre ella, sino un conjunto de relaciones entre dichas unidades. Cuanto más estrechamente interrelacionadas están las relaciones, más organizado es el sistema que forman (A. Rapoport).
  • conjunto de acciones (funciones) vinculadas en el tiempo y el espacio por un conjunto de tareas prácticas de toma de decisiones y de evaluación del comportamiento, es decir, tareas de gestión (S. Sengupta, R. Ackoff).
  • término que se utiliza para designar al menos dos conceptos distintos: un dispositivo regular u ordenado, compuesto de elementos o partes interrelacionados que actúan como un todo; un conjunto o grupo de elementos (partes) necesarios para la ejecución de una determinada operación (A. Wilson, M. Wilson).
  • conjunto no vacío de elementos que contiene al menos dos elementos, cuyos elementos se encuentran entre sí en determinadas relaciones y vínculos (G. Kröber).
  • sistema abstracto o simplemente sistema, que es un conjunto parcialmente conectado de objetos abstractos que son los componentes del sistema. Los componentes del sistema pueden ser orientados o no orientados; su número puede ser finito o infinito; cada uno de ellos puede definirse mediante un número finito o infinito de variables fundamentales (L. Zadeh, Ch. Desoer).
  • conjunto de elementos activos vinculados entre sí (O. Lange).
  • cualquier forma de distribución de la actividad en una cadena, considerada por un observador como regular (G. Pask).
  • conjunto de componentes interrelacionados de una u otra naturaleza, ordenado por relaciones que poseen propiedades perfectamente definidas; este conjunto se caracteriza por una unidad que se expresa en propiedades y funciones integrales del conjunto (V. S. Tyujtin).
  • diversidad de relaciones y vínculos de los elementos de un conjunto que constituye una unidad íntegra. Tiene sentido entender por sistema un conjunto organizado que forma una unidad íntegra (A. D. Ursul).
  • exclusivamente aquel complejo de componentes selectivamente involucrados cuya interacción y relaciones mutuas adquieren el carácter de contribución conjunta de los componentes a la obtención de un resultado útil focalizado (P. K. Anojin).
  • conjunto de elementos reales o imaginarios, delimitados del resto del mundo por cualquier método. Este conjunto es un sistema si: están dadas las relaciones existentes entre dichos elementos; cada uno de los elementos se considera internamente indivisible; con el mundo exterior al sistema, el sistema interactúa como un todo; al evolucionar en el tiempo, el conjunto se considerará un mismo sistema si entre sus elementos en distintos momentos temporales puede establecerse una correspondencia unívoca. La correspondencia debe ser unívoca, no necesariamente biunívoca. El ordenamiento temporal no es un rasgo obligatorio; si hay divergencia, puede considerarse todo como un sistema único o pueden identificarse subsistemas dentro de él (L. A. Blumenfeld).
  • conjunto de objetos en el que se realiza una relación previamente definida con propiedades fijas. La definición dual será la del sistema como conjunto de objetos que poseen propiedades previamente definidas con relaciones fijas entre ellas (A. I. Uyómov).
