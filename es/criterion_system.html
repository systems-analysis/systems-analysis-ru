Criterios de evaluación de sistemas

Los criterios de evaluación de sistemas son reglas o normas que permiten valorar la eficiencia de un sistema, es decir, la correspondencia entre el resultado requerido y el efectivamente alcanzado.

Cuando es posible introducir características cuantitativas y vincular mediante una expresión analítica el objetivo del sistema con los medios para alcanzarlo, dichas expresiones reciben el nombre de criterio de eficiencia, criterio de funcionamiento, función objetivo, etc.
